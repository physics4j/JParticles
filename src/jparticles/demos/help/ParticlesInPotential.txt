<h2>Description</h2>
This applets shows the interaction of particles with a spatial potential.
The spatial potential is generated by superposing gaussian distributions of various means and variances.

<h2>Controls</h2>

<b><i>Viscosity</b></i> : Set the value for the medium viscosity in here.
Don't forget to press the return key to activate the new value.

<p><b><i>Time-step </b></i>: Set the value for the time step in here.
Don't forget to press the return key to activate the new value.</p>

<p><b><i>Space Bar</i></b>: Toggle Stop/Start the animation.</p>
